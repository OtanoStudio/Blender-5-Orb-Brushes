# Blender-5-Orb-Brushes

For blender artists wanting to us the zBrush Orb brushs in blender 5.+, this repo serves as as back up for the original google drive files

These were ported over by [@Careless_School2771](https://www.reddit.com/user/Careless_School2771/) I do not own or claim any rights. They posted them for free use on [reddit post](https://www.reddit.com/r/blender/comments/1t2klok/orb_brushes_for_blender_50/)

---

1. open a command line console and add `git clone https://github.com/OtanoStudio/Blender-5-Orb-Brushes.git`
2. check your users document folder for a Blender folder. windows C:/Users/USER_NAME/Documents/Blender/Assets/Saved/Brushes.
3. If the folder doesn't exist yet open blender.
- Enter sculpt mode.
- in the properties panel select the tool, this will show the current brush selected.
- top right next to the icon of the brush will be an arrow.
- click it and duplicate the brush selected ( I did the draw brush ) and save to assets.
- folder should now exist.
4. copy & paste the .blend files to that location.
5. restart blender and use, they show in the all brushes area of the brush selection.
